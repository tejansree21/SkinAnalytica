📋Overview

About this API

The SkinAnalytica API provides AI-assisted dermoscopy analysis using a three-model ensemble (EfficientNetV2-S + ViT-Large + ConvNeXt-Large). It returns clinical verdicts, ICD-10 codes, NCCN treatment recommendations, and FHIR-compatible reports for each image.

Models

EfficientNetV2-S (AUC 0.9715) · ViT-Large (AUC 0.9887) · ConvNeXt-Large (AUC 0.9883)
Ensemble MelAUC: 0.9609 · pAUC: 0.9316 · Sensitivity: 80.1% · Specificity: 95.5%
Temperature scaling T=0.4095 · Hair removal preprocessing · 7 lesion classes

Classes

mel Melanoma · nv Melanocytic Nevus · bcc Basal Cell Carcinoma · akiec Actinic Keratosis · bkl Benign Keratosis · df Dermatofibroma · vasc Vascular Lesion

🔐Authentication

Current: No authentication required

This version of the API is open for local research use. All endpoints are accessible without any API key or token. For hospital deployment or multi-tenant use, contact the SkinAnalytica team.

🔬Analyze
POST /analyze Single image → full clinical report
Request · multipart/form-data
filebinaryrequiredDermoscopy image (JPEG/PNG/TIFF/BMP, max 20MB)
cancer_typestringoptionalHint: mel, nv, bcc, akiec, bkl, df, vasc
patient_ageintegeroptionalPatient age in years
patient_sexstringoptionalmale / female
patient_idstringoptionalFor linking to patient history
Example · cURL
bash
curl -X POST http://localhost:8001/analyze \
  -F "file=@/path/to/lesion.jpg" \
  -F "patient_age=52" \
  -F "patient_sex=female"
Example · Python
python
import requests

with open("lesion.jpg", "rb") as f:
    res = requests.post(
        "http://localhost:8001/analyze",
        files={"file": f},
        data={"patient_age": 52, "patient_sex": "female"}
    )
data = res.json()
print(data["verdict"], data["confidence"])

POST /analyze/batch Multiple images in one call
Request · multipart/form-data
filesbinary[]requiredArray of dermoscopy images
cancer_typestringoptionalApply same hint to all images
Example · Python
python
files = [("files", open(p, "rb")) for p in image_paths]
res   = requests.post("http://localhost:8001/analyze/batch", files=files)
data  = res.json()
print(f"Flagged: {data['flagged_count']}/{data['total']}")
